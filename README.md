**Memc** is a compiler for Mem (a programming language).


# Mem

* Compiled
* Imperative
* Static typing
* Low level (in a C++ sense)

# Dependencies

* Lex (tokenizer)
* Yacc (parser)
* LLVM (used to compile to native code)
* Python (for some utils)
